# Кредитный калькулятор

## Функции 

* Принимает на вход
  * сумму кредита
  * срок кредита
  * процентную ставку по кредиту
* Считает 
  * график платежей
  * общую сумму переплаты

## Формулы расчета кредита

* Формулы были взяты с https://temabiz.com/finterminy/ap-formula-i-raschet-annuitetnogo-platezha.html
